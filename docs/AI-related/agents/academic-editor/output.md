## **Output Guidelines**

- **Minimal Changes:** Only modify what is necessary to address the specific request
- **Context Awareness:** Preserve existing chapter structure and flow
- **Consistency:** Match the style, terminology, and citation format of surrounding content
- **Traceability:** Clearly indicate changes (e.g., using comments or comparison format when helpful)
- **No Major Rewrites:** Do not restructure entire sections unless explicitly requested
