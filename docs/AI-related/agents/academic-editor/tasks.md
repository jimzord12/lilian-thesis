## **Task Categories**

### **1. Content Insertions**

- **Tables:**

  - Create structured tables summarizing data, comparisons, or classifications
  - Follow academic table format: clear title, numbered (Table X.Y), descriptive column headers
  - Include caption below the table in italics
  - Cite sources where applicable

- **Figures/Image Placeholders:**

  - Suggest appropriate figure placements with descriptive captions
  - Provide detailed figure descriptions that can be used for generation
  - Format: `Figure X.Y: [Descriptive caption]`
  - Include reference markers in the text (e.g., "as shown in Figure 4.2")

- **Side Notes/Callouts:**
  - Add explanatory notes where concepts need clarification
  - Insert "Note:" or "Important:" boxes for emphasis
  - Highlight key insights without disrupting main flow

### **2. Fixes and Corrections**

- **Grammar and Syntax:**

  - Correct tense inconsistencies (use present tense for general facts, past tense for completed studies)
  - Fix sentence structure and punctuation errors
  - Ensure subject-verb agreement

- **Factual Accuracy:**

  - Verify technical claims where possible
  - Correct terminology usage (e.g., ensure MIoT terms are used correctly)
  - Align numbers, dates, and figures for consistency

- **Citation Corrections:**

  - Ensure APA style formatting (Author, Year)
  - Fix missing or inconsistent citations
  - Verify citation placement (typically at end of relevant sentence, before punctuation)

- **Cross-References:**
  - Ensure chapter/section references are accurate
  - Update figure/table references after insertions
  - Check that "see Chapter X" references remain valid

### **3. Polishing and Enhancement**

- **Tone Consistency:**

  - Maintain formal, academic, authoritative tone
  - Remove flowery or colloquial language
  - Ensure objective, impartial phrasing

- **Clarity and Readability:**

  - Break up overly long sentences (>30 words)
  - Improve paragraph transitions
  - Simplify complex constructions without losing precision

- **Technical Precision:**

  - Enhance technical specificity where appropriate
  - Add precise metrics or data points where generic statements exist
  - Use standard industry terminology

- **Formatting:**
  - Ensure consistent heading levels
  - Maintain list formatting consistency
  - Verify proper markdown structure
