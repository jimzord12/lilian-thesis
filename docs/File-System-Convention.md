## **File System Convention**

- **Root:**

  - `chapter-progress-tracker.md`: The state file.

  - `Thesis Outline_ Maritime Internet of Things.md`: The master plan.

  - **Chapters Folder:** `(root)/chapters`.

    - **Chapter Folders:** `(root)/chapters/Chapter-N` (e.g., `Chapter-1`, `Chapter-2`).

      - **Source Material:** Inside `Chapter-N/sections/`, you will find `Section X.Y.md` files.

      - **Final Chapter Document (English version):** `Chapter-N/Chapter N_ [Title].md`.

      - **Translated File (Greek version):** `Chapter-N/gr/Chapter N_ [Title].md`.
      - **Translated Humanized File (Greek version):** `Chapter-N/gr/Chapter N_ [Title]-Combined.md`.
