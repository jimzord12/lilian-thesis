## English Version

| Standard | Organization | Primary Domain | Key Objectives |
|:---|:---|:---|:---|
| **ISO 4891:2024** | ISO | Smart Ship Applications | Standardizes interoperability for smart applications, process management, and compliance levels. |
| **UR E26 / E27** | IACS | Cyber Resilience | Mandates secure system integration (E26) and cyber resilience of onboard equipment (E27). |
| **IEC 61162 Series** | IEC | Digital Interfaces | Defines protocols (NMEA 0183/2000) for maritime navigation and radiocommunication equipment. |
| **FAL Convention** | IMO | Administrative Data | Mandates electronic exchange of reporting formalities via Single Window systems. |
